Number Formatter
=========

A small library that adds commas  to numbers

## Installation

    'npm install @jonas-salomonsson/test'

## Usage

    var numFormatter = require('@jonas-salomonsson/test');
    var formattedNum = numFormatter(35666);

    Output should be '35,666'
